## System at a glance

| Layer | Technology |
|---|---|
| Backend | Spring Boot 3.4.5, Java 17, Spring Cloud 2024.0.1 |
| Frontend | React 18.3, TypeScript 5.4, Ant Design 5.17, Vite 5.2 |
| Databases | PostgreSQL 16, Elasticsearch 8.13, Redis 7 |
| Messaging | Apache Kafka 7.6.0 (KRaft mode) |
| Auth | Keycloak 24.0 (OAuth2/OIDC, PKCE) |
| Storage | MinIO (S3-compatible) |
| Observability | Prometheus, Grafana, ELK Stack |
| Service Discovery | Eureka (HA, 2 peers) |
| API Gateway | Spring Cloud Gateway |

## Services

| Service | Responsibility |
|---|---|
| **api-gateway** | Central routing, JWT validation, rate limiting |
| **task-service** | Tasks, projects, phases, work logs (primary write model) |
| **user-service** | User management via Keycloak Admin API |
| **notification-service** | Email alerts + WebSocket push on task changes |
| **audit-service** | Immutable audit trail of all task changes |
| **reporting-service** | Aggregated hour and task reports, real-time dashboards |
| **search-service** | Full-text task and user search via Elasticsearch |
| **file-service** | File upload/download via MinIO (avatars, attachments) |
| **eureka-server** | Service discovery registry (HA pair) |
